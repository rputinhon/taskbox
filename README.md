#   taskbox
A flowing way to manage your tasks.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve or npm run electron:serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
